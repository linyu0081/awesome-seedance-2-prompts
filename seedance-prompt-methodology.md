# Seedance 2.0 提示词方法论

[第一性原则]

    导演意图优先
        - 导演讲戏本是创意源头，提示词是技术翻译
        - 不要偏离导演的画面意图，不要自行添加导演未描述的内容
        - 如果导演阐述中缺少某些技术细节（如具体运镜），分镜师可以补充，但必须服务于导演的叙事意图

    叙事描述式
        - Seedance 2.0 擅长理解长段叙事性描述
        - 用完整的段落描述，像写电影脚本一样
        - 不要用关键词堆叠，不要用标签式描述

    参考图已见原则
        - Seedance 2.0 能"看到"通过 @引用 传入的参考图
        - 不需要重复描述参考图中已有的静态内容（如人物长相、场景布局）
        - 重点描述"变化"：动作、运动、情绪转变、镜头运动

    有参考图的主角不加外貌形容
        - 通过 @图片 引用了角色参考图的角色，提示词中直接用角色名，不要加额外的外貌形容词
        - 如"清瘦白净的"、"白发苍苍的"、"丹凤眼"、"剑眉星目"等描述一律不写
        - Seedance 2.0 已从参考图获得角色外貌信息，重复描述可能产生冲突
        - 可以描写角色的动作和神态（如"叶辰猛然抬头，目光凌厉"）
        - 群演和一次性配角（无参考图）仍需用文字描述外貌

    一条提示词一个 Shot
        - 每条提示词对应一个 Shot = 一次生成任务，产出一条视频
        - 一个剧情点可以有 1-3 个 Shot（最多 3 次生成，最长 45 秒）
        - 一条提示词内可包含多组镜头（如蒙太奇、连续切换），但叙事上必须连续
        - 不要在一条提示词中塞入叙事上不相关的场景
        - 连续的动作戏和复杂运镜必须放在同一条提示词（同一个 Shot）内

    节拍密度约束
        作用于每个连续镜头内，非整段：
        - 底层规律：1 拍 ≈ 2.5 秒屏幕时间。节拍太密，AI 生成的动作会糊、跳、不自然。
        - 多镜头段落的节拍分布在各镜头切换中，每个镜头各自遵守密度约束，总节拍可超单镜头上限。

    台词密度-时长匹配（重要）
        台词是时长分配中最刚性的约束——说话需要的时间无法压缩。
        
        语速参考：正常 3-4 字/秒，激动 5-6 字/秒，低语 2 字/秒
        
        每个时间段可容纳的台词上限：
        | 可用时长 | 最大台词量 | 适用场景 |
        |---------|-----------|---------|
        | 3s      | ≤ 8 字    | 1 句短台词 + 1 个简单动作 |
        | 5s      | ≤ 15 字   | 1 句中等台词 + 1-2 个动作 |
        | 8s      | ≤ 25 字   | 2 句短台词或 1 句长台词 + 2-3 个动作 |
        | 10s     | ≤ 30 字   | 1 轮简短对话 + 2-3 个动作 |
        | 15s     | ≤ 50 字   | 多轮短对话 + 运镜 + 动作 |

        校验方法：统计 Shot 内所有台词总字数 → 按语速换算朗读时间 → 加上话轮切换停顿（每次 0.5-1s）→ 加上动作/运镜时间 → 总时间必须 ≤ Shot 时长 - 1s（头尾安全区）
        如果超出，必须精简台词或向导演反馈时长不足。

    头尾安全区
        每次 Seedance 生成的前 0.5s 和后 0.5s 为安全区，不放关键内容：
        - 开头：先给环境或主体一个静态建立
        - 结尾：让动作自然收住或保持可衔接的姿态

[平台约束]

    输入限制（单条提示词 / 单次生成任务维度）
        | 输入类型 | 数量上限 | 支持格式 | 大小限制 |
        |---------|---------|---------|---------|
        | 图片    | ≤ 9 张  | jpeg、png、webp、bmp、tiff、gif | 每张 < 30 MB |
        | 视频    | ≤ 3 个  | mp4、mov | 每个 < 50 MB，总时长 2-15s |
        | 音频    | ≤ 3 个  | mp3、wav | 每个 < 15 MB，总时长 ≤ 15s |
        | 文本    | 自然语言提示词 | — | — |
        | **总文件数** | **≤ 12 个** | — | — |

        注意：以上限制是每条提示词（每次 Seedance 生成）的限制，不是全文档限制。
        素材对应表是全文档的总映射表，@图片 总数可以超过 9 张。

    输出参数
        - 生成时长：4-15 秒（可自由选择）
        - 自带音效/配乐
        - 视频总像素数范围：480p（640×640）至 720p（834×1112）

    内容限制
        - 不支持写实真人脸部素材（图片和视频均不可），系统会自动拦截
        - 优先上传对画面或节奏影响最大的素材，合理分配文件数量

[@ 引用语法]

    引用方式
        @图片1    @图片2    @图片3   ...（最多9张）
        @视频1    @视频2    @视频3   （最多3个）
        @音频1    @音频2    @音频3   （最多3个）

    用途指定
        务必明确说明每个引用的作用，不要只写"参考@视频1"而不说参考什么：

        | 用途 | 示例写法 |
        |------|---------|
        | 首帧 | @图片1 作为首帧 |
        | 尾帧 | @图片2 作为尾帧 |
        | 人物形象 | 参考 @图片1 的人物形象 |
        | 场景/背景 | 场景参考 @图片3 |
        | 运镜 | 参考 @视频1 的运镜效果 |
        | 动作 | 参考 @视频1 的动作编排 |
        | 特效 | 完全参考 @视频1 的特效和转场 |
        | 节奏/节拍 | 视频节奏参考 @视频1 |
        | 音色/语气 | 旁白音色参考 @视频1 |
        | 背景音乐 | 背景BGM参考 @音频1 |
        | 音效 | 音效参考 @视频3 的音效 |
        | 服装 | 穿着 @图片2 的服装 |

    多引用组合
        可以在一条提示词中组合多个引用：
        @图片1 的人物作为主角，参考 @视频1 的运镜和动作编排，
        背景BGM参考 @音频1，场景参考 @图片5

    场景引用说明
        scene-prompts.md 中的九宫格是"生成格式"，用于一次性生成所有场景的参考图。
        生成后，用户会将九宫格中的每个格子单独提取出来，作为独立的场景参考图。
        因此在素材对应表中，每个场景必须独立编号为一个 @图片，不能将整张九宫格作为一个 @图片。

    非素材场景的文字补充
        如果某个 Shot 涉及的场景不在素材对应表中（scene-prompts.md 中无该场景的提示词/参考图），
        不使用 @引用，而是参考 scene-prompts.md 中其他场景提示词的写法风格，
        在提示词中用一句话简要补充该场景的关键特征（空间类型、材质/质感、核心视觉元素、氛围基调）。
        示例：叶辰站在一座宽阔的露天石质广场，广场由巨大的青灰色石板铺就，中央矗立着一座三米高的古朴石碑。
        描述要融入叙事段落，具体可视化，不要笼统抽象。

        正确示例（4 个人物 + 9 个场景）：
        | 引用编号 | 素材类型 | 对应素材 |
        |---------|---------|---------|
        | @图片1 | 人物参考 | 林书白 |
        | @图片2 | 人物参考 | 幼年林书白 |
        | @图片3 | 人物参考 | 楚讲师 |
        | @图片4 | 人物参考 | 考核长老 |
        | @图片5 | 场景参考 | 简陋住处（室内）——九宫格格1 |
        | @图片6 | 场景参考 | 青云文院大门——九宫格格2 |
        | @图片7 | 场景参考 | 觉醒仪式大殿——九宫格格3 |
        | ... | ... | ... |

        错误示例：
        | @图片5 | 场景参考 | ep01 场景宫格（整张九宫格）|
        ← 不要这样做，用户无法将整张九宫格上传为一个参考图

[提示词结构]

    基本结构
        1. 素材引用声明（开头）
        2. 场景/环境描述
        3. 主体动作与运镜
        4. 光影/氛围/声音
        5. 风格/技术参数（可选）

    分时段描述（10秒以上推荐使用）
        当生成时长 ≥ 10 秒时，推荐按时间段精确控制画面内容：

        0-3秒：[开场画面描述、运镜、动作]（台词 ≤ 8 字）
        3-6秒：[中段发展]（台词 ≤ 8 字）
        6-10秒：[高潮或关键动作]（台词 ≤ 12 字）
        10-15秒：[收尾、定格画面]（台词 ≤ 15 字）

        分时段描述与台词密度约束配合使用：
        - 每个时间段内的台词字数必须与可用时长匹配（参照台词密度-时长匹配表）
        - 如果某时间段有密集对话，适当延长该段时间或减少台词
        - 时间段之间的衔接要自然
        - 多镜头段落中，每个时间段可对应不同的子场景/镜头
        - 头尾安全区规则仍然适用（首段开头0.5s、末段结尾0.5s）
        - 节拍密度约束仍然适用（每个连续镜头内 1 拍 ≈ 2.5s）

    完整示例
        以@图片1中的林薇为主角，场景参考@图片3的深夜办公室。
        中景，林薇独自坐在工位前，冷白荧光灯照亮她疲惫的面容。
        她揉了揉太阳穴，目光从电脑屏幕缓缓移开，身体靠向椅背。
        视线转向窗外，城市灯火稀落。她轻轻叹了口气，低声自语："又是一个人。"
        镜头从她的侧脸慢慢推向窗外的夜色，整体氛围冷寂孤独，电影感。

    分时段示例（15秒）
        以@图片1中的林薇为主角，场景参考@图片3的深夜办公室。
        0-3秒：远景建立，空旷的办公区只有角落一盏台灯亮着，林薇的剪影映在落地窗上。
        4-8秒：中景推近，林薇揉太阳穴，目光从屏幕移开，身体缓缓靠向椅背，表情从专注转为疲惫。
        9-12秒：特写她的侧脸，视线转向窗外，城市灯火稀落，轻声自语："又是一个人。"
        13-15秒：镜头缓慢拉远，林薇的身影在空旷办公区中越来越小，画面定格。
        音效：键盘声渐弱、空调低鸣、窗外远处的车流声。电影感，冷寂孤独。

[运镜词汇]

    景别
        - 极致特写：仅拍摄眼睛、嘴巴等细节
        - 面部特写：人脸充满画面
        - 中近景：头部和肩部
        - 中景：腰部以上
        - 全景：展示完整人物
        - 远景 / 建立镜头：展示完整环境

    推拉类
        - 推镜头 / 缓慢推近：聚焦、紧张、亲密
        - 拉镜头 / 缓慢拉远：揭示、疏离、结束感
        - 滑动变焦 / 眩晕效果：心理冲击

    横纵摇类
        - 左摇 / 右摇：环顾、跟随水平运动
        - 上摇 / 下摇：强调高度、规模
        - 快速横摇：转场、惊讶

    环绕类
        - 环绕主体：立体感、动感
        - 弧形运动：戏剧性、强调
        - 半环绕（180度）：对话、对峙

    跟拍类
        - 跟拍镜头：跟随主体移动
        - 背后跟拍：跟随主体背影
        - 侧面跟拍：平行跟随
        - 一镜到底：全程无剪辑的连续镜头

    升降类
        - 上升镜头：史诗感、希望
        - 下降镜头：压迫、沉重
        - 航拍镜头：规模、全景

    高级运镜
        - 希区柯克变焦：推拉+变焦的眩晕效果，表现心理冲击
        - 鱼眼镜头：超广角畸变，营造扭曲不安感
        - 低角度仰拍：低机位向上拍摄，增加英雄感/压迫感
        - 俯拍 / 鸟瞰：从高处向下拍摄，全局感
        - 第一人称主观视角：从角色视角出发，沉浸感
        - 快速摇镜：极速水平旋转产生运动模糊
        - 机械臂跟随：多角度灵活跟随人物视线

    手持与固定
        - 手持镜头：真实感、紧张
        - 固定机位：稳定、庄重

[动作词汇]

    面部
        - 缓慢睁眼、微笑浮现、目光转移、眉毛上扬
        - 表情从X转为Y、咬嘴唇、眼眶泛红

    头部
        - 转头、微微歪头、缓慢点头、抬头、低头

    身体
        - 呼吸起伏、前倾、后仰、站起、坐下
        - 向前走、伸出手、拥抱

    手部
        - 举手、握紧、手指动作、比划

    环境互动
        - 头发随风飘动、衣服飘动、水花飞溅
        - 落叶飘落、烟雾升起、尘埃飘浮

[速度修饰词]

    缓慢/轻柔：缓慢、轻柔、渐渐、微妙、细腻
    中等/自然：平稳、稳定、自然、流畅
    快速/强烈：快速、迅速、突然、戏剧性、爆发

[氛围与风格词汇]

    画面风格
        - 电影感、影片质感、大片风格、纪录片风格
        - 电影级质感，胶片颗粒，浅景深
        - 2.35:1宽银幕，24fps
        - 黑白水墨风格 / 动漫风格 / 超写实
        - 高饱和霓虹色调，冷暖对比

    情绪氛围
        - 戏剧性、紧张、悬疑、浪漫、梦幻、怀旧
        - 神秘、欢快、忧郁、暗黑奇幻、仙侠高燃
        - 喜剧风格，表情夸张
        - 温暖治愈、史诗恢宏

[音频设计]

    音频是 Seedance 2.0 的重要能力，提示词中应包含音频指导：

    背景音乐
        - 直接描述风格：背景音乐：恢宏大气 / 紧张悬疑 / 温暖治愈
        - 引用参考：背景BGM参考 @音频1
        - 引用视频中的音乐：音乐风格参考 @视频1

    音效设计
        - 环境音效：走路声、人群声、汽车声、雨声、风声
        - 动作音效：脚步声、呼吸声、衣料摩擦声
        - 情绪音效：心跳声、钟表声、回声

    音画同步
        - 转场画面与音乐节奏卡点
        - 脚步声、呼吸声必须与画面动作贴合
        - 音乐高潮与画面高潮同步

    旁白/对话
        - 角色对白直接写入提示词，用引号标注
        - 可指定音色参考：旁白音色参考 @视频1
        - 可指定语气：哽咽、怒吼、低语、坚定

[视频操作模式]

    除了标准的"从零生成"，Seedance 2.0 还支持以下操作模式，分镜师在需要时可以使用：

    视频延长（向后）
        将 @视频1 延长[时长]。
        [描述延长部分的画面内容]
        注意：生成时长应选择"新增部分"的时长（如延长5秒，生成长度也选5秒）

    视频延长（向前）
        向前延长[时长]，[描述前置画面内容]

    视频编辑
        保留原视频大部分内容，定向修改特定元素：
        - 角色替换：将 @视频1 中的角色换成 @图片1 的角色
        - 元素添加：在 @视频1 的基础上添加 [新元素]
        - 剧情修改：在 @视频1 的基础上改变 [具体情节]

    视频融合
        将多个视频片段衔接融合：
        @视频1 的画面逐渐过渡到 @视频2，[描述过渡方式]

[从导演讲戏到提示词的转化步骤]

    1. 提取素材引用
        - 从导演阐述的人物和场景字段，对应 character-prompts.md 和 scene-prompts.md 中的素材
        - 在提示词开头声明引用，明确每个引用的用途

    2. 提取核心动作
        - 从导演阐述中识别主要动作链
        - 保留具体物理动作，删除抽象描述

    3. 提取运镜意图
        - 从导演阐述中识别镜头运动描述
        - 转化为规范的运镜语言
        - 如果导演未明确指定运镜，根据情绪和叙事目的补充

    4. 提取声音/台词
        - 从导演阐述中提取对白、自语、环境音
        - 保持原文台词不变
        - 补充音效设计和背景音乐指导

    5. 台词密度校验（必须执行）
        - 统计该 Shot 内所有台词总字数
        - 按语速估算朗读时间 + 话轮切换停顿
        - 加上动作/运镜时间
        - 验证总时间 ≤ Shot 时长 - 1s（头尾安全区）
        - 如果超出：精简台词 → 转画外音 → 删减次要台词 → 标注"⚠ 时长紧张"

    6. 判断是否使用分时段描述
        - 时长 ≥ 10 秒：推荐使用分时段描述
        - 时长 < 10 秒：使用连续段落描述即可

    7. 整合为完整提示词
        - 按照提示词结构组织
        - 确保叙述流畅、连贯
        - 控制在合理长度（与时长匹配）
        - 确保每个时间段的台词量不超过该段可用时长的容量
        - 检查每个 @引用 都有明确的用途说明

    8. 追加画面质量后缀
        - 在提示词末尾统一追加固定文案："画面稳定流畅，无闪烁无重影，人物结构正常，动作自然不僵硬，4K 高清细节清晰。不要生成字幕。"
        - 该后缀所有 Shot 统一使用，不需要根据剧情内容调整

[禁忌规则]

    绝对禁忌
        - 否定句（"不要移动" → "保持静止"）
        - 重复参考图中已有的静态内容
        - 抽象概念（"体现自由" → 用具体动作表达）
        - 一条提示词中包含叙事上不相关的场景
        - @引用不指定用途（"参考@视频1" → "参考@视频1的运镜效果"）
        - 上传写实真人脸部素材

    避免事项
        - 过长提示词（控制信息密度与时长匹配）
        - 违反物理规律的动作
        - 与参考图内容矛盾的描述
        - 指令冲突（同时要求"固定镜头"和"环绕镜头"）
        - 单个连续镜头内节拍过密
        - 素材上传了但提示词中未用 @ 标注用途
        - 台词量超出时长容量（必须按校验公式计算，台词是最刚性的时长约束）
        - 在 3-5 秒的短时间段内塞入多轮对话（3 秒只够说 8 个字）

[优质提示词分类型写作方法论]

    以下方法论从 935 条社区验证的优质 Seedance 2.0 提示词中归纳总结而来，
    按视频类型分为 12 个类别，每个类别提供：核心结构、关键要素、写作范式和注意事项。

    ═══════════════════════════════════════════
    一、提示词五大结构模式（通用）
    ═══════════════════════════════════════════

    模式 A：分镜头/时间码结构（推荐用于 10s+ 复杂场景）
        按秒数精确控制画面：
        [00:00-00:05] Shot 1：[场景 + 动作 + 镜头 + 特效]
        [00:05-00:10] Shot 2：[场景 + 动作 + 镜头 + 特效]
        [00:10-00:15] Shot 3：[场景 + 动作 + 镜头 + 特效]
        适用：动作战斗、变形机甲、广告商业、音乐 MV
        优势：时间轴精确、画面可控、适合多镜头切换

    模式 B：连续叙事段落（推荐用于 5-15s 叙事场景）
        用完整段落像写电影剧本一样连续描述：
        "空中超现实的战场：漂浮的岩石岛屿在雷暴中漂移…蒙面浪人冲过漂浮的平台…
        镜头在岛屿之间飞速穿梭…"
        适用：史诗叙事、奇幻冒险、情感场景、一镜到底
        优势：叙事流畅自然、Seedance 2.0 理解力强

    模式 C：标签分段式（推荐用于需要精确控制多维度的场景）
        用方括号标签组织信息层次：
        【核心主题】描述
        【人物设定】外貌 / 服装 / 状态
        【场景环境】地点 / 天气 / 氛围
        【氛围与画质】摄影机 / 色调 / 风格
        【运镜】镜头模式 / 运动轨迹
        【时间轴】分秒描述
        适用：变形序列、机甲组装、中文武侠/仙侠
        优势：维度清晰、不遗漏关键信息

    模式 D：极简指令式（适用于 4-8s 简单场景）
        用一到两句话高度概括：
        "一辆豪华轿车变形成擎天柱，在东京雨夜的背景下与哥斯拉搏斗，
        爆发出爆炸和能量冲击波。"
        适用：概念验证、简单动作、单一事件
        优势：简洁高效、留给模型充分发挥空间
        注意：效果随机性较大，不适合需要精确控制的场景

    模式 E：JSON/结构化数据式（新兴趋势）
        用 JSON 格式组织提示词参数：
        {"style": "...", "scene": "...", "shots": [...], "audio": "..."}
        适用：需要极精确参数控制的动画风格和商业广告
        优势：参数无歧义、适合批量化生产

    ═══════════════════════════════════════════
    二、分类型写作方法论
    ═══════════════════════════════════════════

    ────────────────────────────────────────
    类型 1：动作 / 战斗场景（占比最大，约 35%）
    ────────────────────────────────────────

    核心结构
        场景设定 → 对峙/蓄力 → 动作爆发 → 高潮碰撞 → 结局定格

    关键要素
        - 速度感词汇：极速、瞬间、猛烈、爆裂、shockwave、instantly、explosive
        - 物理反馈描写：地面炸裂、碎石飞溅、冲击波扩散、装甲碎裂、尘土飞扬
        - 动作连贯性：明确描述动作链的前后因果，不要跳跃
        - 镜头跟随动作：镜头运动要配合打斗节奏（快速横摇、360°环绕、子弹时间）
        - 能量特效：色彩 + 形态 + 运动方式 + 物理影响

    写作范式
        推荐使用分镜头时间码结构（模式 A）或连续叙事段落（模式 B）
        时长建议：12-15 秒，分 3-4 个 Shot

    优质示例要素
        ✓ 具体的招式名和技能效果（如"万剑诀"、"水之呼吸"、"雷之闪光"）
        ✓ 描述打击的物理后果（地面裂开、树木折断、能量波扩散）
        ✓ 每个 Shot 有明确的镜头运动指令
        ✓ 末尾追加画质标签：8K UHD、cinematic、Hollywood VFX
        ✓ 音效指示：金属碰撞声、能量爆炸声、无对白

    典型范例（中文，分时段）
        真人漫改 · 呼吸法对决（15 秒 · 超燃特效版）
        【风格】：好莱坞真人漫改电影质感，暗黑武士风，4K 超清，极致快剪，粒子光效炸裂
        【场景】：月光下的迷雾森林，泥泞地面，落叶
        [00:00-00:05] 镜头 1：年轻武士月光下沉下重心，拔刀出鞘，蓝色水龙凭空出现围绕身体旋转
        [00:05-00:10] 镜头 2：金发剑士化作金色闪电残影，Z 字形高速突进，地面炸裂
        [00:10-00:15] 镜头 3：正面碰撞，蓝色水龙与金色闪电炸裂，形成能量风暴，画面在极致光芒中结束

    典型范例（英文，连续叙事）
        A surreal battlefield in the sky: floating rock islands drifting through a thunderstorm.
        The masked ronin dashes across the drifting platforms, pursued by a colossal winged beast.
        The camera hurtles from island to island, rapid handheld cuts capture the ronin leaping
        impossible gaps, sword carving arcs of light. The finale shows the camera diving behind
        him as he rides a bolt of lightning into the monster's chest vortex.
        720p, 16:9, 15 seconds

    注意事项
        - 战斗类节拍密度较高，建议每 3-5 秒一个关键动作节点
        - 多人战斗要明确每个角色的位置和动作
        - 避免同时发生太多独立事件，模型可能无法全部渲染

    ────────────────────────────────────────
    类型 2：变形 / 机甲 / 超级英雄变身
    ────────────────────────────────────────

    核心结构
        人物初始状态 → 触发事件 → 分阶段变形过程 → 完成形态展示 → 力量释放

    关键要素
        - 变形过程按身体部位逐步描述：前臂 → 胸甲 → 肩甲 → 腿部 → 头盔 → 翅膀/武器
        - 机械细节词汇：活塞伸展、齿轮咬合、装甲板锁定、能量管路点亮、磁性锁扣
        - 声效极重要：金属咔嗒声、活塞排气嘶嘶声、装甲锁定沉重咚声
        - 强烈推荐一镜到底（single shot, one take, no cuts）
        - 能量效果：蓝色能量管路、紫色光芒、电磁脉冲、电弧

    写作范式
        推荐使用标签分段式（模式 C）+ 时间码
        时长建议：12-15 秒

    优质示例要素
        ✓ 详细的变形时间轴（0-3s 坠落/3-6s 机甲汇聚/6-9s 组装/9-12s 成形/12-15s 着陆）
        ✓ 模拟专业摄影设备（IMAX 胶片摄影机、Panavision C 系列镜头、焦段 35mm、光圈 f4）
        ✓ FPV 第一人称跟随 + 轨道环绕 + 低角度英雄仰拍
        ✓ 明确变形前后的对比状态
        ✓ 结尾定格的"英雄着陆姿态"（hero landing pose）

    典型范例
        【核心主题】写实科幻机甲变身，自由落体中组装
        【人物基础设定】穿深蓝色紧身飞行服的男性特工
        【核心机制】机甲零件从四面八方高速飞来，在坠落过程中依次拼装
        【场景环境】万米高空，云层翻涌，地面城市轮廓可见
        【氛围与画质】IMAX 胶片质感，Panavision C 系列镜头（35mm, f4）
        【运镜】一镜到底，FPV 第一人称跟随 → 轨道环绕 → 低角度仰拍
        【时间轴】
        0-3s：高空坠落，衣物猛烈飘动，面部表情坚毅
        3-6s：远处金属零件高速飞来汇聚，形成前臂护甲
        6-9s：胸甲合拢、肩甲锁定、腿部装甲包裹，机械咔嗒声密集
        9-12s：头盔闭合，HUD 点亮，完整机甲成形，能量管路蓝光流动
        12-15s：机甲调姿，喷射器启动，超级英雄着陆姿态

    注意事项
        - 变形顺序要符合物理直觉（从四肢到躯干，或从躯干向外扩展）
        - 声效是变形类的灵魂，必须描写机械运动的声音
        - 一镜到底效果最佳，避免切镜头打断变形过程

    ────────────────────────────────────────
    类型 3：广告 / 商业展示
    ────────────────────────────────────────

    核心结构
        产品入场 → 拆解/展示 → 组装/呈现 → 360° 展示 → 品牌收尾

    关键要素
        - 产品质感词汇：glossy、sleek、polished、reflective、carbon fiber、精工细作
        - 时间码精确到 2 秒级别（00:00-00:02, 00:02-00:04…）
        - 光影是核心：高对比聚光灯、戏剧性镜头光晕、全息网格、丁达尔光
        - 镜头运动平滑优雅：smooth 360° orbit、macro close-ups、cinematic slider
        - 可包含旁白脚本（NBC 播音风格 / 高端品牌腔调）

    写作范式
        推荐使用分镜头时间码（模式 A），时间码精度到 2 秒
        时长建议：10-15 秒

    优质示例要素
        ✓ 产品登场方式有创意（从天而降、旋转飞入、爆炸拆解）
        ✓ 微距特写展示材质细节（碳纤维纹路、金属反光、玻璃通透）
        ✓ 环境极简或高端（黑色背景 + 聚光灯 / 未来感展厅）
        ✓ 末尾定格在产品全貌 + 品牌标识
        ✓ 音效：机械精密声、优雅的低音 drop

    典型范例
        High-End Commercial Ad — AURORA FIZZ
        00:00-00:02 | 产品从画面上方高速旋转飞入，金属光泽反射聚光灯
        00:02-00:04 | 产品水平分离为三层，内部结构全息展示
        00:04-00:07 | 层层细节微距特写，气泡从内部升起，折射出彩虹光芒
        00:07-00:08 | 三层磁性吸附重新组合，锁定声清脆利落
        00:08-00:10 | 360° 慢速环绕展示完整产品，背景渐变为品牌色

    婚摄广告子类
        - 按场景分段，每段指定摄影风格品牌（Vogue editorial / Valentino / Dior）
        - 色彩调性：warm white and gold、creamy white + cool pink、orange-gold gradient
        - 注重光线的柔和度和质感（自然光 / 金色逆光 / 窗光）

    注意事项
        - 广告类最忌"平淡"，每个 2 秒段都需要有视觉冲击
        - 产品始终是画面焦点，不要让背景抢戏
        - 品牌调性一致性极其重要

    ────────────────────────────────────────
    类型 4：日常生活 / 情感 / 浪漫
    ────────────────────────────────────────

    核心结构
        场景氛围铺设 → 人物状态建立 → 微妙互动 → 情绪转折/高潮 → 情感定格

    关键要素
        - 节奏缓慢细腻：slowly、gently、softly、渐渐、微微、缓缓
        - 微表情描写是灵魂：睫毛投下细微阴影、瞳孔微微放大、喉结滚动、耳垂泛红
        - 对话要包含自然停顿：200-400ms 停顿、轻轻咬下唇、呼吸同步
        - 环境音为主导：夏蝉、纸笔摩擦声、低频心跳脉冲、轻盈钢琴
        - 浅景深 + 暖色调 + 胶片颗粒感

    写作范式
        推荐使用连续叙事段落（模式 B）或分时段（模式 A），节奏要留白
        时长建议：10-15 秒

    优质示例要素
        ✓ 微表情变化有层次（从 X 到 Y：从专注到惊讶，从惊讶到羞涩）
        ✓ 具体的对话文本和语气标注
        ✓ 环境细节烘托情绪（窗外的城市灯火、飘落的樱花、升腾的咖啡蒸汽）
        ✓ 色温指定：warm golden / cherry blossom pink / cold moonlight blue
        ✓ 镜头运动极缓：推近面部、焦点转移（rack focus）

    典型范例（中文）
        以 @图片1 中的林薇为主角，场景参考 @图片3 的深夜办公室。
        0-3 秒：远景建立，空旷的办公区只有角落一盏台灯亮着，林薇的剪影映在落地窗上。
        4-8 秒：中景推近，林薇揉太阳穴，目光从屏幕移开，身体缓缓靠向椅背，
        表情从专注转为疲惫。
        9-12 秒：特写她的侧脸，视线转向窗外，城市灯火稀落，
        轻声自语："又是一个人。"
        13-15 秒：镜头缓慢拉远，林薇的身影在空旷办公区中越来越小，画面定格。
        音效：键盘声渐弱、空调低鸣、窗外远处的车流声。电影感，冷寂孤独。

    典型范例（英文，浪漫微距）
        15-second cinematic Japanese drama...
        0-8s: Side-by-side desks, the boy's hand pauses mid-writing, pencil trembling
        slightly, eyelashes cast subtle shadows on cheeks under warm afternoon light...
        9-15s: Extreme close-up of both faces, the girl slowly turns, first showing
        dazed surprise, then shyly lowers her head for 0.3 seconds, gently biting her
        lower lip, cheeks and earlobes bloom with cherry blossom pink...

    注意事项
        - 情感类忌节奏太快，每个情绪变化需要 2-3 秒的呈现时间
        - 微表情描写不能太密——一个 5 秒段最多写 2 个微表情变化
        - 对话量要严格遵守台词密度约束，情感类台词通常语速较慢（2-3 字/秒）
        - 声音设计以环境音和轻音乐为主，避免喧闹

    ────────────────────────────────────────
    类型 5：奇幻 / 神话 / 仙侠
    ────────────────────────────────────────

    核心结构
        宏大场景建立 → 神话元素呈现 → 力量觉醒/法术释放 → 史诗高潮 → 壮阔定格

    关键要素
        - 东方美学特有元素：丁达尔光穿透云层、衣袂飘飘、剑气纵横、灵力涌动
        - 西方奇幻特有元素：巨龙喷火、魔法阵、光之裂隙、暗黑能量
        - 环境规模宏大：万丈山崖、云海翻涌、古堡巍峨、星空璀璨
        - 特效华丽：能量漩涡、光芒万丈、元素觉醒（风火水土）
        - 音乐恢宏大气：交响乐、史诗管弦乐、汉斯·季默风格

    写作范式
        推荐使用连续叙事段落（模式 B）或标签分段式（模式 C）
        时长建议：12-15 秒

    优质示例要素
        ✓ 仙侠四灵/五行/八卦等文化符号的具体视觉化
        ✓ 法术释放有完整的蓄力-释放-扩散过程
        ✓ 人物渺小 vs 环境宏大的对比（建立镜头）
        ✓ 色彩方案：青白色灵力、金色佛光、暗紫邪气
        ✓ 镜头语言：航拍远景建立 → 推近主体 → 环绕法术效果

    典型范例
        史诗级中国神话电影：
        远景建立——云海之上的悬浮仙山，瀑布倒流飞向天空，丁达尔光从云层缝隙中倾泻。
        镜头缓慢推近山巅的修炼者，白衣飘飘，盘膝悬浮于虚空之中。
        他猛然睁眼，双瞳金光闪烁，周身灵力化为四道光柱（青龙、白虎、朱雀、玄武），
        螺旋上升冲破云层。
        航拍俯冲跟随光柱上升，最终在云层之上展开为巨大的四灵阵法，光芒照耀整片天际。
        背景音乐：恢宏交响乐，鼓声渐强。

    注意事项
        - 仙侠/奇幻类场景宏大但要聚焦核心视觉点，不要同时铺太多元素
        - 法术效果需要描写完整过程（蓄力→释放→影响范围），不能只说"释放法术"
        - 丁达尔光、云海、衣袂飘动是中文提示词的高频高效元素

    ────────────────────────────────────────
    类型 6：音乐 / 舞蹈 / MV
    ────────────────────────────────────────

    核心结构
        风格设定 + BGM 指定 → 分段表演（Intro → Verse → Dance Break → Climax）→ 定格结束

    关键要素
        - BGM 风格必须明确指定：Trap 电子、808 鼓、K-pop 复古、Chinese electronic
        - 服装描写极其详细（MV 的核心视觉元素）
        - 舞蹈动作要具体：hip-hop bounce → street dance freeze → body wave → quick footwork
        - 镜头与节奏（beat）同步：低角度仰拍 + 360° 环绕 + 快速剪切
        - 色调鲜明：霓虹紫蓝冷色调 / 暖黄胶片色调 / 高饱和度

    写作范式
        推荐使用分镜头时间码（模式 A），与音乐节奏卡点
        时长建议：12-15 秒

    优质示例要素
        ✓ 歌词/Rap 词直接写入提示词中
        ✓ 服装颜色、品牌、配饰明确（如"黑皮夹克 + 白色印花 T 恤胸前大黑字 YOLO"）
        ✓ 舞蹈动作与镜头切换卡 beat
        ✓ 每个分镜头标注摄像机角度变化
        ✓ 色彩滤镜/调色方案

    典型范例
        16:9 横屏，街头说唱 MV 风格，霓虹紫蓝冷色调。
        人物：80 岁老奶奶，银色短发，黑皮夹克 + 白色印花 T 恤 + 黑色工装裤 + 白色高帮鞋。
        BGM：Trap 电子音乐，重型 808 鼓。
        [Rap 歌词] "八十岁的腿，蹦得比你高！银发飘飘，这是我的骄傲！…"
        0-5s：低角度仰拍，老奶奶从街角走出，节奏感十足地打响指
        5-10s：360° 环绕拍摄，她做出 street dance freeze 姿势
        10-15s：快速剪辑，多角度捕捉 body wave + footwork，BGM 高潮同步

    注意事项
        - 舞蹈类提示词的音画同步极其重要，镜头切换要与 beat 对齐
        - 歌词不宜太长，遵守台词密度约束
        - 服装描写是 MV 的关键区分度，不能省略

    ────────────────────────────────────────
    类型 7：自然 / 动物 / 纪录片
    ────────────────────────────────────────

    核心结构
        环境全景铺设 → 动物行为建立 → 关键动作（捕猎/迁徙/互动）→ 结果 → 远景收尾

    关键要素
        - 摄影风格：natural handheld documentary camera、shallow DOF、cinematic slow motion
        - 自然光为主导：golden sunrise、early morning mist、moonlight through canopy
        - 环境音效真实：风声、水流、鸟鸣、脚步声踩在雪地上
        - 旁白风格明确：BBC 风格 / National Geographic 风格 / 温暖叙述者
        - 相比其他类型更加简洁，不需要复杂分镜

    写作范式
        推荐使用连续叙事段落（模式 B），简洁自然
        时长建议：10-15 秒

    优质示例要素
        ✓ BBC 风格旁白文本直接写入（如 "Beneath the silent blanket of snow, life continues unseen..."）
        ✓ 动物微行为描写（呼吸形成小白雾、耳朵微微转动、瞳孔收缩）
        ✓ 环境氛围描写与动物行为的对比（寂静环境 vs 突然的捕猎爆发）
        ✓ 镜头跟随动物运动轨迹
        ✓ 光线变化标注（dawn golden light → overcast → sunbeam through clouds）

    典型范例
        超写实野生动物纪录片。
        黎明时分，一只红狐沿着雪地森林边缘谨慎移动，呼吸在冰冷的空气中形成小白雾。
        它突然竖起耳朵，身体低伏，前爪悬在半空——
        猛然跃起，头部精准地扎入雪层中，叼出一只田鼠。
        镜头从侧面跟拍到正面特写，慢动作展示跃起的弧线。
        自然光，浅景深，电影级慢动作。
        旁白（BBC 风格平静语气）："在寂静的雪毯之下，生命悄然延续。而狐狸，已掌握了倾听的艺术…"

    注意事项
        - 纪录片类应避免拟人化或超自然描写，保持写实
        - 旁白语气要与画面节奏匹配
        - 自然光描写比人工布光更重要

    ────────────────────────────────────────
    类型 8：恐怖 / 悬疑
    ────────────────────────────────────────

    核心结构
        安静铺垫 → 异响/线索 → 试探/探索 → Jump Scare 揭示 / 悬念留白

    关键要素
        - 精确的三段式结构（三个 Shot，每个约 4 秒）
        - 光影是核心手段：dim moonlight、flashlight、flickering lights、heavy shadows
        - 节奏控制——前慢后快：slowly walks down → suddenly lunges out
        - 镜头语言制造恐惧：Medium shot slowly approaching → Sudden violent close-up jump scare
        - 最精炼的类型——用最少文字制造最大张力

    写作范式
        推荐使用分镜头时间码（模式 A），严格三段式
        时长建议：10-12 秒

    优质示例要素
        ✓ 环境铺垫占总时长的 60%（前 6-7 秒），Jump Scare 只占最后 3-4 秒
        ✓ 声音设计的"静→响"对比（寂静→突然的尖叫/撞击）
        ✓ 特写+快推是 Jump Scare 的经典镜头组合
        ✓ 反面角色的揭示只给部分特征（苍白的脸、一只手、闪烁的轮廓）
        ✓ 结尾可以留悬念（画面黑屏+声音延续）

    典型范例
        【风格】经典砍杀恐怖，暗黑卧室氛围，悬疑节奏，突然的恐怖揭示。
        [00:00-00:04] Shot 1：噪音。昏暗卧室，仅月光照明。
        女孩被异响惊醒，缓缓坐起，目光投向床下的黑暗。
        镜头：中景缓慢推近。
        [00:04-00:08] Shot 2：探索。她探出头，弯腰往床下看。
        手电筒光颤抖地照向黑暗。什么也没有——或者说，看起来什么也没有。
        镜头：低角度，手电筒光是唯一光源。
        [00:08-00:12] Shot 3：Jump Scare。一张苍白的面孔突然出现在距镜头几英寸处，
        从床下猛然伸手抓住她。
        镜头：突然暴力特写跳吓。

    注意事项
        - 恐怖类切忌铺垫太短直接吓人，效果会大打折扣
        - 声音设计是恐怖感的 50%，"安静"比"喧闹"更恐怖
        - 不要描写过于血腥的内容，Seedance 有内容限制

    ────────────────────────────────────────
    类型 9：体育运动 / 极限运动
    ────────────────────────────────────────

    核心结构
        运动员/场景建立 → 准备动作 → 关键技巧执行 → 慢动作回放 → 结果/着陆

    关键要素
        - 物体追踪镜头：camera locks onto the golf ball、camera tracks directly behind the spinning sphere
        - 极限场景叠加：暴风雨 + 悬崖 + 雪崩 + 高速
        - 物理细节：轮胎变形、泥土爆溅、粉笔尘雾、肌肉线条
        - FORMAT 标签：15s / 6 SHOTS / men's apparatus final / no dialogue
        - 慢动作 + 正常速度的交替使用

    写作范式
        推荐使用分镜头时间码（模式 A）或连续叙事（模式 B）
        时长建议：12-15 秒

    优质示例要素
        ✓ 多角度镜头切换展示同一动作（正面/侧面/背后/俯拍/主观视角）
        ✓ 慢动作用在关键瞬间（击球/翻腾/落地/碰撞）
        ✓ 环境极端条件增加戏剧性（暴雨/大雪/悬崖/高空）
        ✓ 结尾的特写定格（球入洞/完美落地/冲过终点）
        ✓ 体育纪录片风格旁白可选

    典型范例
        15 秒极限场景。暴雨中，暴风骤起的海边悬崖球场。
        高尔夫球手身穿亮黄色雨衣，挥杆击球——
        镜头锁定高尔夫球飞行轨迹，紧紧跟随旋转的球体穿越驱动的雨帘。
        雨滴在球体表面碎裂成微小水花，球的旋转清晰可见。
        慢动作：球划过最高点，背景是闪电照亮的乌云。
        终极特写：球撕裂湿草，溅起泥土弧线，精准落入球洞。
        cinematic, 4K, film grain

    注意事项
        - 体育类核心在于"运动轨迹的可视化"，要让镜头跟着运动物体走
        - 慢动作使用要克制，全程慢动作会失去张力
        - 物理细节（水花、泥土、粉尘）是真实感的关键

    ────────────────────────────────────────
    类型 10：动画 / 定格动画 / 2D 手绘
    ────────────────────────────────────────

    核心结构
        风格声明（最重要）→ 材质/技术说明 → 角色描述 → 动作叙事 → 风格收尾

    关键要素
        - 风格声明必须反复强调：Pure 2D. Hand-drawn. No 3D. Ever. / 定格动画，手工材质
        - 2D 手绘关键词：drawn on ones、24fps、hand-painted watercolor、soft watercolor textures
        - 定格动画关键词：handcrafted tactile materials、real fabric、clay figures、stop-motion
        - 物理夸张表达：exaggerated squash and stretch、neck stretching comically、eyes spinning
        - 材质触感描写：cloth physics、hand-drawn wind lines、纸质纹理

    写作范式
        推荐使用连续叙事（模式 B），辅以 [cut] 标记分镜
        时长建议：10-15 秒

    优质示例要素
        ✓ 开头和结尾都强调风格约束（"Pure 2D, No 3D" 放在首句和末句）
        ✓ 描述材质而非效果（"hand-painted watercolor backgrounds" 而非 "beautiful backgrounds"）
        ✓ 动画夸张手法的具体描写（squash & stretch 的程度、timing）
        ✓ 定格动画注重故事叙事性（常有寓言式小故事）
        ✓ 用 [cut] 或 [smash cut] 标记动画剪辑点

    典型范例（定格动画）
        定格动画，手工粘土材质，真实布料纹理。
        全景：一个火柴盒头粘土人和一个点燃蜡烛头粘土人坐在真实的微型公园长椅上。
        [cut] 远处传来隆隆声，两个头同时转向。
        [cut] 特写：一个灭火器头粘土人骑着微型摩托车滚入画面…
        全程保持手工定格动画质感，可见指纹痕迹，12fps 抖动，微型实景道具。

    典型范例（2D 手绘）
        Pure hand-drawn 2D cartoon animation. NO 3D elements, NO CGI effects.
        A young witch on a tiny broomstick zips through a twilight sky full of floating islands.
        Tight rear-follow shot, the camera barely keeping up.
        She weaves between clockwork windmills, squeezes through a flock of startled owls,
        and narrowly dodges a sleepy dragon's tail.
        Style: hand-drawn 2D, soft watercolor textures, warm sunset colors,
        Ghibli-inspired whimsical energy. 24fps, drawn on ones.

    注意事项
        - 动画类最常见的问题是 Seedance 生成 3D 而非 2D——必须多次强调风格约束
        - 定格动画要描写"手工感"（指纹、布料褶皱、微型道具）
        - 帧率标注很重要：2D 常用 24fps drawn on ones，定格用 12fps

    ────────────────────────────────────────
    类型 11：旅行 / Vlog / 生活方式
    ────────────────────────────────────────

    核心结构
        地标设定 → 人物出场（穿搭描写）→ 互动动作 → 场景转换 → 下一地标

    关键要素
        - 男友视角 / 旅行博主自拍是核心视角
        - 每个场景对应不同穿搭（服装变化体现地域特色）
        - 场景转换方式：rotation match cut、seamless landmark transitions
        - 按地标分段：0-3s Paris → 3-6s New York → 6-9s Tokyo → 9-12s Santorini
        - 风格参考：Wes Anderson 对称构图 / Influencer fashion reel / Cinematic travel vlog

    写作范式
        推荐使用分镜头时间码（模式 A），按地标分段
        时长建议：12-15 秒

    优质示例要素
        ✓ 每个地标都有标志性视觉元素（埃菲尔铁塔/时代广场/威尼斯运河）
        ✓ 人物与环境的互动（走路/跳跃/回头/挥手/品尝美食）
        ✓ 穿搭变化增加丰富度
        ✓ 转场方式创意化（旋转匹配剪辑/360° spin/走过画面边缘）
        ✓ 明亮高饱和的色彩方案

    典型范例
        以 @图片1 的角色为主角，男友视角手持拍摄。
        0-3s：巴黎，埃菲尔铁塔前，她穿法式碎花裙转身微笑，
        牵起镜头（男友）的手向前走去。
        3-6s：旋转匹配剪辑——转到纽约时代广场，她换成牛仔外套+墨镜，
        在人群中兴奋地张开双臂。
        6-9s：再次旋转——威尼斯运河边，她穿白色长裙，
        呆呆地望着经过的白马，阳光洒在水面上。
        9-12s：旋转——东京涩谷十字路口，她穿和风元素混搭装，
        在行人中回头给镜头比心。
        色调：明亮高饱和，旅行博主风格，influencer fashion reel energy。

    注意事项
        - 旅行类的核心是"地标可识别性"，场景描写要有标志性元素
        - 穿搭变化太多会增加生成难度，4 个场景 4 套已是上限
        - 转场方式要一致（全用旋转或全用走过），避免混用

    ────────────────────────────────────────
    类型 12：喜剧 / 幽默 / 创意短片
    ────────────────────────────────────────

    核心结构
        正常建立 → 意外/反转 → 夸张反应 → 荒诞升级 → 抖包袱/定格

    关键要素
        - 反转是喜剧的核心——建立期望然后打破
        - 表情夸张：eyes wide、jaw drops、double take、over-the-top reaction
        - 节奏感：铺垫要稳，反转要快，反应要停留足够长让观众反应过来
        - 背景音乐/音效配合笑点：sudden silence、comic sound effect、dramatic sting
        - 可以混搭风格（如"港式无厘头 + 电影级画质"）

    写作范式
        推荐使用连续叙事（模式 B）或分时段（模式 A）
        时长建议：10-15 秒

    优质示例要素
        ✓ 铺垫阶段看似正常严肃，反转突然且荒诞
        ✓ 角色反应是笑点的放大器，需要足够的画面时间
        ✓ 声效在笑点前突然安静，笑点后夸张配乐
        ✓ 可以包含梗/网络文化参考
        ✓ 结尾定格在最荒诞的瞬间

    典型范例
        港式无厘头喜剧，电影级画质。
        0-4s：中景，古装朝堂上，大臣们正襟危坐，皇帝面色凝重——
        "朕今日有一件大事宣布。"
        5-8s：满朝文武屏住呼吸，镜头在紧张的面孔间快速横摇——
        9-12s：皇帝站起，从袍子里掏出一只活鸭：
        "朕的鸭走丢了，谁看见了？"
        12-15s：满朝文武集体石化，一个大臣的帽子自动掉落。
        画面定格在鸭子"嘎"的一声中。

    注意事项
        - 笑点前的铺垫不能太长，也不能太短——约占总时长的 40-50%
        - 喜剧的声音设计非常重要（静默→笑点→夸张音效）
        - 角色表情反应需要 1-2 秒的停留时间让观众消化

    ═══════════════════════════════════════════
    三、跨类型通用写作规律
    ═══════════════════════════════════════════

    提示词长度与效果的关系
        | 长度 | 字数 | 效果特征 | 适用场景 |
        |------|------|---------|---------|
        | 极简 | 1-2 句 | 模型自由发挥，效果随机 | 概念验证、简单事件 |
        | 中等 | 100-300 字 | 性价比最高，可控且有创意空间 | 大部分场景 |
        | 详细 | 300-600 字 | 精确控制，每秒有交代 | 复杂动作、商业广告 |
        | 极详细 | 600+ 字 | 最精确但模型可能无法完全执行 | 变形序列、多镜头大制作 |

    七层黄金公式（优质提示词的通用框架）
        1. [格式/参数] 时长、宽高比、分辨率
        2. [风格定义] 整体视觉风格、电影参考、色调
        3. [场景设定] 环境、时间、天气、氛围
        4. [角色设定] 外貌、服装、表情、状态
        5. [分时段动作] 按时间码详细描述动作 + 镜头运动
        6. [特效/光影] 能量效果、物理效果、光影方案
        7. [风格标签] 末尾 3-8 个关键词锚定风格

    高频高效的画质锚定标签
        通用：cinematic, 4K, film grain, shallow DOF
        动作类：8K UHD, Hollywood VFX, Dolby Vision HDR, ultra high detail
        情感类：warm tones, soft lighting, film grain, 35mm
        广告类：premium quality, studio lighting, high contrast
        纪录片：natural lighting, documentary style, BBC quality
        动画类：hand-drawn, 24fps, watercolor texture, Ghibli-inspired
        仙侠类：东方美学，丁达尔光，衣袂飘飘，电影级调色

    中英文提示词风格差异
        | 维度 | 中文提示词 | 英文提示词 |
        |------|-----------|-----------|
        | 结构偏好 | 方括号标签分段 / 时间码 | 连续叙事段落 |
        | 详细程度 | 普遍更长更结构化 | 从极简到极详细均有 |
        | 擅长题材 | 武侠/仙侠/古风/变身 | 科幻/奇幻/好莱坞动作 |
        | 特色元素 | 丁达尔光、衣袂飘飘、剑气 | volumetric fog, Hollywood VFX |
        | 禁止标签 | 常用"禁止字幕/水印/LOGO" | 较少使用禁止标签 |
        | 技术参数 | 侧重画质和风格指定 | 侧重镜头运动描述 |

    通用禁止标签（建议末尾追加）
        中文：禁止生成字幕、LOGO、水印。画面稳定流畅，无闪烁无重影。
        英文：No subtitles, no watermarks, no text overlays. Stable, flicker-free footage.
